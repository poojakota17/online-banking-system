# Team-project-artemis - Online Banking System

## Local Development
### How to run the application
````
cd go to the project destination folder /cmpe202
````
````
mvn spring-boot:run
````

### How to test the API
http://localhost:8080/swagger-ui.html

### Database lookup
http://localhost:8080/h2-console
